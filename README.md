# Power BI Sales Dashboard

## Project Overview
This project presents an interactive Sales Dashboard created using Power BI to analyze sales performance, profit trends, regional sales, and customer insights.

The dashboard helps businesses monitor key performance indicators and make data-driven decisions.

---

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Visualization
- DAX Measures

---

## Key Insights
- Total Sales: 56.70K
- Total Profit: 36.05K
- Profit Margin: 63.58%
- Best Selling Product: Laptop
- North region generated highest sales.

---

## Dashboard Features
- KPI Cards (Sales, Quantity, Profit, Margin)
- Sales Trend Analysis
- Product Performance
- Regional Sales Comparison
- Top Customers Analysis
- Gender-wise Sales Distribution
